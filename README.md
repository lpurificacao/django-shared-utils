# django-shared-utils

A collection of shared Django utility functions for development and debugging.

## Features

### `patch_queryset_pretty_sql()`

Monkey-patches Django's `QuerySet` to add a `pretty_sql()` method that prints the underlying SQL query in a formatted, readable way using [sqlparse](https://github.com/andialbrecht/sqlparse).

This is useful during development and debugging to quickly inspect the SQL generated by the Django ORM.

## Requirements

- Python 3.x
- Django
- [sqlparse](https://pypi.org/project/sqlparse/)

## Installation

Clone the repository and ensure it's on your Python path:

```bash
git clone <repo-url> django-shared-utils
```

Install the dependency:

```bash
pip install sqlparse
```

## Usage

Call `patch_queryset_pretty_sql()` once at startup (e.g. in a Django shell session or your project's `AppConfig.ready()` method):

```python
from utils import patch_queryset_pretty_sql

patch_queryset_pretty_sql()
```

After patching, every `QuerySet` instance gains a `.pretty_sql()` method:

```python
from myapp.models import Article

qs = Article.objects.filter(status="published").order_by("-created_at")
qs.pretty_sql()
```

Output:

```sql
SELECT "myapp_article"."id",
       "myapp_article"."title",
       "myapp_article"."status",
       "myapp_article"."created_at"
FROM "myapp_article"
WHERE "myapp_article"."status" = 'published'
ORDER BY "myapp_article"."created_at" DESC
```
